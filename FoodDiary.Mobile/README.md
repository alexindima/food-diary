# FoodDiary.Mobile

Capacitor mobile shell for the FoodDiary Angular client.

The shell packages the existing Angular app from `../FoodDiary.Web.Client` and hosts native bridges for mobile-only health integrations:

- Android: Health Connect
- iOS: HealthKit, when added later

## Layout

```text
FoodDiary.Mobile/
  capacitor.config.ts
  package.json
  android/              # generated by Capacitor
  ios/                  # generated later when iOS support is added
```

## Workflow

```powershell
npm install
npm run build:web
npm run sync:android
npm run build:android:debug
```

Android builds require JDK and Android Studio/SDK.

On Windows, `npm run build:android:debug` runs Gradle through `scripts/android-gradle.ps1`.
The script sets `JAVA_HOME`, `ANDROID_HOME`, and a temporary ASCII drive mapping for repositories stored under paths with non-ASCII characters.
