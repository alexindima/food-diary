Notes:

- This checklist is extracted from the Google Docs text export.

Verification pass 2026-06-20:

- Environment: `http://localhost:4200/`, user `admin@fooddiary.club`, Chromium via Playwright.
- In-app Browser plugin was attempted first, but its tab/webview connection became unstable, so regular Playwright was used for the verification pass.
- Google Sign-In on localhost logged expected 403/FedCM errors; email/password login worked and these Google errors did not block tested flows.
- Browser-specific reports for Yandex Browser remain unverified in this pass.
